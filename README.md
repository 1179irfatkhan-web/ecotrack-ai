# EcoTrack AI

Carbon Footprint Awareness Platform

## Features

- Carbon Calculator
- Sustainability Dashboard
- Carbon Reduction Simulator
- PDF Report Generation
- User Authentication
- AI Sustainability Coach

## Tech Stack

- Django
- Bootstrap 5
- Chart.js
- SQLite
- ReportLab

## Setup

pip install -r requirements.txt
